Copy and Paste:

JDE has a built-in copy and paste feature that allows you to copy and paste the contents of a file to the keyboard along with smaller snippets of code. The large snippet copying can be achieved by clicking on the copy button whereas the small snippet copying can be achieved by use of the Command + C combinations. Pasting can be achieved by use of the Command + V combination
