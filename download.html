How To Download / Use:

To install start by downloading and then unzipping the file. From here open the dmg and move over the app into the applications folder. Then close the dmg and move it to the trash. Now go into the applications folder and double click the app to open it or right click and press open. You may get a warning that the app could not be checked by Apple for malicious software or "“App-Store” can’t be opened because it is from an unidentified developer." If you get this, read on, if not, congratulations! The reason this warning appears is because I did not notarize the app with Apple. From here you can click the mac icon in the top right corner of your screen and go into system preferences. Now go into Security & Privacy. From here go into the general tab and in the "Allow apps downloaded from:" section you can see the application. Click the open anyway button next to it and the app should open in a moment. This process will not need to be repeated. If there is no open anyway button or the app does not show please contact me using the suggestions button on the home screen.

NOTE:

Python must be installed for the terminal to run properly and even then it may be required to download command line tools from xcode. Another issue with the terminal is that it does have issues running files though this will be fixed in a future update
